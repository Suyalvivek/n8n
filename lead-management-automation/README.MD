# Lead Management Automation using n8n

Automated lead collection and qualification workflow built with n8n.

## Features

- Lead capture through form
- Store leads in Google Sheets
- Budget-based lead qualification
- Automatic email notifications
- Personalized follow-up emails
- Lead status tracking

## Workflow

1. User submits inquiry form
2. Lead data stored in Google Sheets
3. Budget validation performed
4. Qualified leads trigger notification email
5. Service-specific email sent to customer
6. Lead status updated automatically

## Tech Stack

- n8n
- Google Sheets
- Gmail API
- Webhooks
- Conditional Logic

## Use Cases

- Marketing Agencies
- Freelancers
- Lead Generation Teams
- Digital Agencies

## Import

Import `workflow.json` into n8n and connect your credentials.
